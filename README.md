# AI Resume Screener

Upload resumes and a job description — ranks the best candidates instantly.

🔗 Live Demo: https://ai-resume-screener-zdls.onrender.com

---

## Built With

- Python + Flask
- OpenAI GPT-4o
- HTML + CSS + JavaScript
- SQLite + PostgreSQL
- Deployed on Render

---

## Features

- Upload multiple PDF resumes
- Instant candidate ranking
- Score chart and PDF report
- Login and screening history
- Works without API key
